Python – Renombrar un archivo

Inicio/Python – Renombrar un archivo
Python – Renombrar un archivoadmin2018-10-02T05:20:55-06:00

Para renombrar un archivo en Python 3 sólo debes usar la función rename del módulo os.  El código es realmente sencillo.

import os
archivo = "/home/decodigo/Documentos/python/archivos/archivo.txt"
nombre_nuevo = "/home/decodigo/Documentos/python/archivos/archivo_renombrado.txt"
os.rename(archivo, nombre_nuevo)
import os archivo = "/home/decodigo/Documentos/python/archivos/archivo.txt" nombre_nuevo = "/home/decodigo/Documentos/python/archivos/archivo_renombrado.txt" os.rename(archivo, nombre_nuevo)
import os

archivo = "/home/decodigo/Documentos/python/archivos/archivo.txt"
nombre_nuevo = "/home/decodigo/Documentos/python/archivos/archivo_renombrado.txt"

os.rename(archivo, nombre_nuevo)

La función pide como parámetros la ruta completa del archivo y la ruta completa del archivo con el nuevo nombre.  Si existe un archivo con el mismo nombre, lo sobrescribirá.



Al ejecutar el código el archivo se renombra.

Esperamos que el ejemplo sea de utilidad.
